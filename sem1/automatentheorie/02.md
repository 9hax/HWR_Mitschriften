## Automaten (DEA)

Ausgewählte Operatoren:

* ``|`` - Selektion (oder)
* ``*`` - Wiederholung
* ``ø`` - Nulloperator
* ``€`` - Einsoperator
* ``()``- Klammern

Rechenregeln:
 
( ø | a ) = a  
( ( a | b ) | ø ) = ( a | ( b | ø ) )
